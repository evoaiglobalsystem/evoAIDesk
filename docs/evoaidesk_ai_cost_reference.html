What does "Daily Volume" mean?
Daily Volume is the number of times that AI action is triggered per day. For example, Lead Qualification = 50 means 50 new leads are processed by AI each day. The monthly cost is simply:
Cost per action × Daily Volume × 30 days = Monthly Cost
e.g. OpenAI Lead Qual: $0.0001 × 50 × 30 = $0.15/month. Adjust these volumes to match your actual expected usage.

Recommended Strategy for EvoAIDesk

Phased approach to maximise quality while keeping costs predictable

Phase 01 — Validate

Use Gemini Flash for MVP

Free tier covers most validation workloads. Zero cost to test lead qualification, tagging, and basic chat flows before committing to production providers.

Phase 02 — High-Volume Ops

Claude Haiku for automation

At $1/$5 per MTok, Haiku is the cheapest production-grade model. Ideal for tagging, lead scoring, and routing where volume is high but quality demands are moderate.

Phase 03 — Customer Facing

OpenAI or Claude Sonnet for chat

Both deliver excellent quality for customer-facing chat. Claude Sonnet is ~50% cheaper than GPT-4o for the same output quality at typical EvoAIDesk conversation lengths.

Phase 04 — Cost Optimisation

Batch API + Prompt Caching

Enable Claude Batch API (50% off) for async jobs like CRM summaries and reports. Add prompt caching for shared system prompts to save up to 90% on input tokens.

🎯
Bottom line: A full EvoAIDesk workload (all 5 agents, typical SMB volume) costs ~$4.75/month on Claude Haiku, ~$17.50 on Sonnet, and ~$22.84 on GPT-4o. All well within the $20/month initial budget — leaving room to scale.

📐
How to use this table: For any new agent you design, count your expected Input Tokens (X) — the prompt + context you send — and Output Tokens (Y) — the response you expect back. Then apply the formula:
Cost = (X × input rate + Y × output rate) ÷ 1,000,000
Rule of thumb: ~1 token ≈ 0.75 words ≈ 4 characters. A 300-word prompt ≈ 400 tokens.
