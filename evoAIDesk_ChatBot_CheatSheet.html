Limitations to Know
🔤
Use exact keywords
"Who's my star closer?" won't work. Use "top performer" instead. The bot matches fixed phrases, not free-form language.
👤
Full rep names only
Use the rep's full name as stored in the CRM. "Chandra" won't match "Chandrashekhar".
⏱️
Data is cached (60 min)
The bot caches data to stay fast. Type "refresh" if you need real-time data after a recent CRM update.
📏
Lead lists capped at 15 by default
All lead list queries show up to 15 results. Add "all" to see everything: "show all leads" or "show all hot leads". For ranked queries (top deals, latest, oldest) the default is top 5 unless you specify a number: "latest 10 leads".
🔁
One query at a time
The bot processes one message at a time. Wait for the response before sending the next query.
🚫
Off-topic queries blocked
The bot only handles CRM queries. Anything unrelated returns a polite error with example commands.
