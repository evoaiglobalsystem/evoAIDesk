How permissions work
READ
All roles can view leads and the activity timeline, but only within their own workspace. Data is never visible to other organisations.
CREATE
Admin and Manager can add new leads. Agents have read-only access and must ask an admin or manager to create leads on their behalf.
UPDATE
Admin and Manager can edit any lead field — status, assignment, follow-up date — and soft-delete or restore leads via the Trash.
DELETE
Admin only. Permanently removes a lead. This is irreversible. Managers can only soft-delete (move to Trash), not permanently erase.

All dates and timestamps — follow-ups, created dates, activity logs — display in your local business timezone. Configured per tenant so distributed teams stay aligned.

Personalise automated emails so they come from your business address (e.g. hello@yourbusiness.com) — not a generic sender. Fully configurable via Google Email Client settings per tenant for a professional sender identity on every automation.
